### 문제 설명

<p>다음과 같은 수열을 <strong>균형잡힌 수열</strong>이라고 정의하자:</p>

<ul>
<li>길이가 $1$인 모든 수열은 균형잡힌 수열이다.</li>
<li>길이 $2k+1$의 수열 $S = [S_0, \dots, S_{2k}]$가 다음 조건을 만족하면 균형잡힌 수열이다.
<ul>
<li>$[S_0, S_1, \dots , S_{k-1}]$이 균형잡힌 수열이다.</li>
<li>$[S_{k+1}, S_{k+2}, \dots , S_{2k}]$가 균형잡힌 수열이다.</li>
<li>$S_k$는 수열 $S$의 모든 원소 중 <strong>유일한</strong> 최댓값이다.</li>
</ul>
</li>
</ul>

<p>$N$개의 정수로 이루어진 수열 $A$가 주어진다. $A[i \dots j]$는 수열 $A$의 $i$번 원소부터 $j$번 원소까지로 구성된 길이가 $j-i+1$인 수열이다. 예를 들어 $A = [3,5,7,2,9]$일 때, $A[1 \dots 3]$은 $[5,7,2]$이고, $A[4 \dots 4]$는 $[9]$이다.</p>

<p>$Q$개의 쿼리가 주어진다. 각 쿼리는 수열의 특정 원소를 변경하는 연산이다. 이 연산은 누적된다. 초기 상태와 각 쿼리를 수행한 이후, $0 \le i \le j \le N - 1$이며 $A[i \dots j]$가 균형잡힌 수열인 정수 두 개의 쌍 $(i,j)$의 총 개수를 구하여라.</p>

 Empty

 Empty
